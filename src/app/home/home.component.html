Demostración de plataforma

Esta aplicación web es un prototipo que muestra parte de la funcionalidad diseñada para solucionar el problema del hackaton baja digital. Cabe destacar que esta plataforma esta delimitada a Equipos Medicos de Alta Tecnologia (EMAT)

Este es un pequeño prototipo que no debe ser usado en ambiente productivo. Así notará que solamente muestra características muy básicas.

De cualquier manera es una manera perfecta de demostrar la idea en acción.

La documentación la puede encontrar en el siguiente enlace https://github.com/jose-zama/baja/wiki

Y el código fuente aquí https://github.com/jose-zama/baja

Uso

Del lado izquierdo encontrara el menu con cuatro opciones:

Inicio Lo traera de vuelta a esta pagina.
Invertir Muestra el flujo en el que un usuario puede invertir en un EMAT
Reservar Muestra el flujo en el que un usuario pueden agendar una cita para el uso de un EMAT
Estadisticas Muestra como se puede mostrar los datos obtenidos apartir del uso del EMAT
